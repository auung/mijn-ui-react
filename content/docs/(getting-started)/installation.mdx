### Configure TypeScript Path Aliases

Since this project is still in a beta phase, and extensive testing is ongoing, we recommend configuring TypeScript with path aliases for better project organization and readability. Please use the following setup in your tsconfig.json:

```json title="tsconfig.json"
  {
    compilerOptions: {
      "baseUrl": ".", // Sets the root directory                   // [!code highlight]
        "paths": {    // Path aliases for cleaner imports          // [!code highlight]
          "@/*": ["./src/*"],           // [!code highlight]
          "@mijn-ui/*": ["./mijn-ui/*"] // [!code highlight]
      }
    }
  }
```

### Recommended Folder Structure

This setup will help keep your code modular and organized, especially for complex projects. Here’s the suggested folder structure to work seamlessly with the path aliases:

### Configure tailwind

Since our components are stored in a separate `mijn-ui` folder, we need to include this folder in the Tailwind CSS content array. This ensures Tailwind scans these files to generate the required CSS classes:

```js title="tailwind.config.js"
/** @type {import('tailwindcss').Config} */

export default {
    content: [
    ...
    "./mijn-ui/**/*.{js,ts,jsx,tsx,mdx}", // Ensure Tailwind scans all mijn-ui files  // [!code highlight]
  ],
  theme: {
    extend: {
      colors: {
        main: {
          DEFAULT: "hsl(var(--main) / <alpha-value>)",
          text: "hsl(var(--main-text) / <alpha-value>)",
          border: "hsl(var(--main-border) / <alpha-value>)",
        },

        surface: {
          DEFAULT: "hsl(var(--surface) / <alpha-value>)",
          text: "hsl(var(--surface-text) / <alpha-value>)",
        },

        neutral: {
          DEFAULT: "hsl(var(--neutral) / <alpha-value>)",
          text: "hsl(var(--neutral-text) / <alpha-value>)",
        },

        accent: {
          DEFAULT: "hsl(var(--accent) / <alpha-value>)",
          text: "hsl(var(--accent-text) / <alpha-value>)",
        },

        primary: {
          DEFAULT: "hsl(var(--primary) / <alpha-value>)",

        },

        secondary: {
          DEFAULT: "hsl(var(--secondary) / <alpha-value>)",

        },
```
