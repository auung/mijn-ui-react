### Install Next.js and Required Dependencies

```bash title="bash"
  npx create-next-app@latest
```

Since we are in the beta stage, the following options work pretty well for installation. Please ensure to select these options:

```bash title="bash"
  ...
  ✔ Would you like to use TypeScript? Yes
  ✔ Would you like to use ESLint?  Yes
  ✔ Would you like to use Tailwind CSS? Yes
  ✔ Would you like your code inside a `src/` directory? Yes
  ✔ Would you like to use App Router? (recommended) Yes
  ✔ Would you like to use Turbopack for next dev? Yes
  ✔ Would you like to customize the import alias (@/* by default)? No
```

  After setting up the Next.js project, install these dependencies used by Mijn-UI components:

  ```package-install
  react-icons class-variance-authority tailwindcss-animate clsx tailwind-merge
  ```

</Step>
<Step>

### Configure TypeScript Path Aliases

Since this project is still in a beta phase, and extensive testing is ongoing, we recommend configuring TypeScript with path aliases for better project organization and readability. Please use the following setup in your tsconfig.json:

```json title="tsconfig.json"
  {
    compilerOptions: {
      "baseUrl": ".", // Sets the root directory                   // [!code highlight]
        "paths": {    // Path aliases for cleaner imports          // [!code highlight]
          "@/*": ["./src/*"],           // [!code highlight]
          "@mijn-ui/*": ["./mijn-ui/*"] // [!code highlight]
      }
    }
  }
```

### Recommended Folder Structure

This setup will help keep your code modular and organized, especially for complex projects. Here’s the suggested folder structure to work seamlessly with the path aliases:

<Files>
  <Folder name="public">
      <File name="..." />
  </Folder>
  <Folder name="src">
    <Folder name="app">
      <File name="page.tsx" />
      <File name="..." />
    </Folder>
  </Folder>
  <Folder defaultOpen name="mijn-ui">
    <Folder name="components">
      <File name="button.tsx" />
      <File name="..." />
    </Folder>
    <Folder name="utils">
      <File name="index.tsx" />
      <File name="..." />
    </Folder>
    <Folder name="context">
      <File name="UnstyledProvider.tsx" />
      <File name="..." />
    </Folder>
    <Folder name="types">
      <File name="index.ts" />

### Add Themes

These are the default themes provided by mijn-ui. If you would like to create your own theme, learn more about themes [here](/docs/theming).


```json doc-gen:file
{
  "file": "./content/common/component-themes.css",
  "codeblock": {
    "lang": "css",
    "meta": "title=\"global.css\""
  }
}
```

</Step>

<Step>
### Configure tailwind

Since our components are stored in a separate `mijn-ui` folder, we need to include this folder in the Tailwind CSS content array. This ensures Tailwind scans these files to generate the required CSS classes:

 ```json doc-gen:file
{
  "file": "./content/common/tw-next-example.config.js",
  "codeblock": {
    "lang": "js",
    "meta": "title=\"tailwind-config.js\""
  }
}
```

</Step>

    <Step>
      ### Include Essential Utilities, Types, Context, and Hooks and Install Dependencies

    1. Within the `mijn-ui` folder, create a `utils` directory and add the following file and install necessary dependencies:

      ```json doc-gen:file
      {
        "file": "./mijn-ui/utils/index.tsx",
        "codeblock": {
          "lang": "tsx",
          "meta": "title=\"mijn-ui/utils/index.tsx\""
        }
      }
      ```
    2. Within the `mijn-ui` folder, create a `types` directory and add the following file: 

      ```json doc-gen:file
      {
        "file": "./mijn-ui/types/index.ts",
        "codeblock": {
          "lang": "ts",
          "meta": "title=\"mijn-ui/types/index.ts\""
        }
      }
      ```
    3. Create a `context` folder, and add:

      ```json doc-gen:file
      {
        "file": "./mijn-ui/context/UnstyledProvider.tsx",
        "codeblock": {
          "lang": "tsx",
          "meta": "title=\"mijn-ui/context/UnstyledProvider.tsx\""
        }
      }
      ```
    4. Create a `hooks` folder, and add:

      ```json doc-gen:file
      {
        "file": "./mijn-ui/hooks/use-controlled-state.ts",
        "codeblock": {
          "lang": "ts",
          "meta": "title=\"mijn-ui/hooks/use-controlled-state.ts\""
        }
      }
      ```
    </Step>
