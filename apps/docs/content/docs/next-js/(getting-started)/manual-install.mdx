### Requirements

- [NextJS 15](https://nextjs.org) or later
- [React 19](https://react.dev) or later
- [Tailwind Css 3.4](https://tailwindcss.com/) or later

### Starter Template

Manual setups can be time-consuming, so I created a [nextjs-starter](https://github.com/mijn-ui/nextjs-template) template to help you get started quickly you can either use this or follow the steps [below](#setting-up-manually) to integrate Mijn-UI components manually.

---

### Setting Up Manually

<Steps>
  <Step>
  **Install Next.js and Required Dependencies**

  Begin by creating a new Next.js application 

  ```bash title="bash"
  npx create-next-app@latest
  ```

  Then, add the required Mijn UI packages:

  ```package-install
  @mijn-ui/react-utilities @mijn-ui/react-hooks @mijn-ui/react-theme
  ```
  </Step>

  <Step>
  **Configure TypeScript Path Aliases**

  ```json doc-gen:file
  {
    "file": "./examples/react/_configs/tsconfig-example.json",
    "codeblock": {
      "lang": "json",
      "meta": "title=\"tsconfig.json\""
    }
  }
  ```
  **Recommended Folder Structure**

  I recommend using this folder structure for manual integration, similar to the packages folder structure. This way, you won't need to update any component paths, as the source codes are from the mijn-ui packages folder.

  <Files>
    <Folder name="public">
      <File name="..." />
    </Folder>
    <Folder name="src">
      <Folder name="app">
        <File name="page.tsx" />
        <File name="..." />
      </Folder>
    </Folder>
    <Folder defaultOpen name="mijn-ui">
      <File name="react-accordion.tsx" />
      <File name="react-button.tsx" />
      <File name="react-input.tsx" />
      <File name="..." />
    </Folder>
  </Files>

  <Alert
    title=""
    description="Note: This folder structure is a suggestion. You can customize it according to your project requirements."
    className="mt-4"
  />
  </Step>


  <Step>
  **Configure Tailwind Theme Plugin:**

    `@mijn-ui/react-theme` is a Tailwind CSS plugin that provides the default theme for Mijn-UI components. Currently, only one theme is available, but you can customize it by following the [override](/docs/next-js/theming#overriding-colors) guide.

    ```tsx title="tailwind.config.js"
    import { mijnUiPreset } from "@mijn-ui/react-theme"
    /** @type {import('tailwindcss').Config} */

    export default {
      content: [
        ...
        "./mijn-ui/**/*.{js,ts,jsx,tsx}", // Ensure Tailwind scans all mijn-ui components  // [!code highlight]
      ],
      presets: [mijnUiPreset], // [!code highlight]
    }
    ```
  </Step>
  <Step>
  **That's it**

  You can now start adding components to your project. Check out the [components](/docs/next-js/components/accordion) section to learn more about the available components.
  </Step>
</Steps>
